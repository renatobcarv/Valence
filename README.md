# Valence - Revenue Recovery AI Platform

Valence is a SaaS platform that helps companies detect and recover lost revenue opportunities inside their sales funnel using AI.

## Architecture
- **Frontend/API**: Next.js App Router (apps/web)
- **Database/Auth**: Supabase (PostgreSQL)
- **AI Services**: Modular AI analysis (packages/ai)
- **Styling**: TailwindCSS

## Quick Start
1. `npm install`
2. Configure `.env` from `infra/env/.env.example`
3. `npm run dev`
